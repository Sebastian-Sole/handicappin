# UX Evaluation Persona

Review from the perspective of the user test evaluators. The user tests account for 25% of the total contract evaluation — this is not optional polish, it's a scoring criteria. Consider task completion rate, error recovery, learnability, and whether the interface communicates its capabilities clearly. Review against the specific test tasks (A-1 through A-9 for staff, L-1 through L-8 for public). Norwegian language quality matters.
