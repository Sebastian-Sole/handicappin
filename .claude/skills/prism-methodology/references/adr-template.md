# Architecture Decision Record Template

Use when exploration surfaces a decision point. Save to `decisions/adr-{NNN}-{slug}.md` in the session directory.

## Template

```markdown
# ADR-{NNN}: {Title}

## Status

Proposed | Accepted | Deprecated | Superseded by ADR-{XXX}

## Context

What problem or question prompted this decision? What constraints exist?

## Options Considered

### Option A: {Name}
- Description
- Pros
- Cons

### Option B: {Name}
- Description
- Pros
- Cons

## Decision

What we decided and why. Reference specific findings from the exploration.

## Consequences

What becomes easier and what becomes harder because of this decision.

## Sources

Links to findings, documentation, and evidence that informed this decision.
```
